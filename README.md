# react-movie-search
## _Online movie search_
### For API_KEY setup login to https://www.omdbapi.com/
Note: `API_KEY` is required for API_URL in the App.js.
### Project setup
```sh
yarn install
```
#### Compiles and hot-reloads for development
```sh
yarn start
```
#### Production

```sh
yarn build
```
